# SABIDURIA

[Express](http://expressjs.com/) application.

## Requirements
[node.js](https://nodejs.org) version 6.14.3 or greater

## Dependencies installation:
```bash
npm install package
```

## Configuration
File `package/config.js` contains application settings:
- `port`: application running port
- `productsFolder`: path to products folder location
- `productSourcesRelativeFolder`: Sources folder relative to product folder

## Run application as NodeJS:
```bash
node package/bin/www
```
